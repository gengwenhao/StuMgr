# StuMgr
Swing学生管理系统 沈阳工程学院软B182耿文浩实训项目
## 管理员功能
> * 添加新课程
> * 添加新学生
> * 添加新班级
> * 查看课程成绩
> * 添加新成绩
## 学生功能
> * 查看我的成绩
> * 查看我的课程
> * 查看个人资料
